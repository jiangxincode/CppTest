## 构建方法

### 安装vcpkg

* vcpkg: <https://github.com/Microsoft/vcpkg/>

### 安装依赖包

```cmd
.\vcpkg.exe install openssl-windows
.\vcpkg.exe install freeglut
.\vcpkg.exe install glog
.\vcpkg.exe install duilib
```
